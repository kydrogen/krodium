# Installation Steps

## Prerequisites
1. Install miniconda:
   ```
   1. curl https://repo.anaconda.com/miniconda/Miniconda3-latest-Windows-x86_64.exe -o .\miniconda.exe
   2. start /wait "" .\miniconda.exe /S
   3. del .\miniconda.exe
   ```
2. Install VSCode python extension
3. In VSCode, set Python interpreter to Conda
